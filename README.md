# fundizen-backend
Spring Boot backend for Fundizen – a full-stack crowdfunding platform using Firebase Auth and MongoDB.

# Fundizen Backend

🚀 Fundizen is a full-stack crowdfunding platform. This is the **Spring Boot backend**, integrated with:
- 🔐 Firebase Authentication (Token + Email Verification)
- 📦 MongoDB Atlas (NoSQL database)
- ☁️ Firebase Admin SDK
- ✉️ Optional Email notifications (Spring Mail)

---

## 🛠️ Tech Stack

- **Spring Boot 3**
- **MongoDB Atlas**
- **Firebase Auth & Admin SDK**
- **Java 17**
- **Deployed on DigitalOcean or Railway (optional)**

---

## 🧩 Features

- User Registration + Firebase Email Verification
- Firebase ID Token-based Login
- Admin role validation
- Campaign creation, listing, donation tracking
- Optional: Email service and image upload
